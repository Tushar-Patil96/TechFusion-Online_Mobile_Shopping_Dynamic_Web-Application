# TechFusion-Online_Mobile_Shopping_Dynamic_Web-Application
Developed a dynamic eCommerce website for mobile phones with user and admin panels, product management, secure user authentication, cart and wish-list functionality, and an integrated order management system. Built using PHP, MySQL, HTML, CSS, and JavaScript.

How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL from the XAMPP Control Panel
3. Copy the project folder into the htdocs directory
#C:\xampp\htdocs\

4. Open phpMyAdmin and create a database
5. Import the provided .sql file into the database
6. Update database credentials in the PHP config file (if required)
7. Open your browser and run:
#http://localhost/project-folder-name/
